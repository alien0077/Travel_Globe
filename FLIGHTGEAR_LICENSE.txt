Derived Data
------------
The file shared/offline-packs/aviation/regions/global.airgraph.json is a
compiled/transformed version of awy.dat, containing only waypoints that
form the main connected graph. The compilation is original work but the
underlying data remains GPL v2.

Other flight navigation data in this project comes from OurAirports
(Public Domain / CC0).
